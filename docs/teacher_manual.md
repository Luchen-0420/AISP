# AISP 实训平台 - 教师操作指南

## 1. 账号注册与登录

### 1.1 注册账号
1.  访问注册页面 `/register`。
2.  在顶部切换栏选择 **“我是老师”**。
3.  填写以下信息：
    *   **用户名**: 登录账号。
    *   **密码**: 登录密码。
    *   **真实姓名**: 您的实际姓名。
    *   **工号**: 学校教职工号。
4.  点击“注册并登录”。

---

## 2. 班级管理 (Classes)

### 2.1 创建班级
1.  进入 **“班级管理”** 页面。
2.  点击右上角 **“创建班级”**。
3.  输入班级名称（如：2024级临床医学1班）和描述。
4.  **获取邀请码**: 创建成功后，班级卡片上会显示一个 **6位数字邀请码**。

### 2.2 学生管理
*   **分发邀请码**: 将邀请码告知学生。学生注册时输入该码，即可自动加入该班级。
*   **查看学生**: 点击班级卡片，可查看已加入的学生列表。
*   **删除班级**: 将鼠标悬停在班级卡片上，点击右上角的红色删除图标可删除班级（慎用，将解绑所有学生）。

---

## 3. 病例库与 AI 生成 (Case Library)

### 3.1 创建标准病例 (Template)
1.  进入 **“病例库”** 页面。
2.  点击 **“新建标准病例”**。
3.  填写疾病名称（如：高血压）、科室（如：心内科）及基础描述。
4.  保存后，该病例将作为“母版”。

### 3.2 AI 生成变体 (Variant Generation)
1.  在病例库列表中，找到刚创建的标准病例。
2.  点击 **“生成变体” (Generate Variant)**。
3.  **配置参数**:
    *   **难度**: 选择 初级 / 中级 / 高级。
    *   **依从性**: 病人配合程度。
    *   **性格**: 如 焦虑、温和、暴躁。
4.  点击 **“开始生成”**。AI 将根据母版和参数，生成一个具体的虚拟病人（包含姓名、年龄、具体主诉、现病史等）。
5.  **预览与保存**: 确认生成内容无误后，点击 **“保存变体”**。

### 3.3 发布任务
*   一旦变体被“保存”，它对关联班级的所有学生即刻可见。
*   学生端刷新后即可看到该任务并开始训练。
